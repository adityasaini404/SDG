# 💧 Clean Water Watch

A web application to support UN SDG 6 – Clean Water and Sanitation.  
Users can report water quality issues, view reported problems, and access educational tips.

## Features
- Report an issue with location, type, and description
- View all reported issues
- Educational tips about clean water
- Data saved in browser (localStorage)

## How to Run
1. Download all files (index.html, style.css, script.js)
2. Open index.html in any browser
or
